# Умножение квадратных матриц

## Описание

Программа на C++ для перемножения квадратных матриц с автоматической верификацией результатов с помощью Python (NumPy).

## Структура проекта

| Файл | Назначение |
|------|------------|
| `main.cpp` | Основной код C++ (умножение матриц, замер времени, подсчёт операций) |
| `verify.py` | Скрипт для верификации результатов через NumPy |
| `generate.py` | Генерация тестовых матриц |

Также в работе использовались 3 файла данных:

- `a.txt` — исходная матрица A
- `b.txt` — исходная матрица B
- `c_cpp.txt` — результат умножения, полученный C++ программой

## Верификация результатов

### Скрипт `verify.py`

Выполняет проверку корректности умножения матриц путём сравнения с эталоном от NumPy.

**Особенности:**
- Читает данные из `a.txt`, `b.txt` и `c_cpp.txt`
- Поддерживает матрицы разных размеров 
- Для каждого размера выполняется сравнение результатов
